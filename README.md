# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 진행 목록
* 사용자로 부터 입력 받는 자동차의 이름 예외 처리
    * 5글자 이하 확인
    * 쉼표로 구분 확인
* 이동 로직 생성
    * 4 이상은 전진
    * 3 이하는 정지
* 자동차 객체 구현
* 0에서 9 사이의 랜덤 값 생성
* 사용자로부터 이동 횟수 입력
* 자동차 경주 생성
* 우승자 선정 후 발표
